
ProAct AI 🤖💡

Link : https://proact-ai-150095457740.asia-southeast1.run.app/

An AI-driven automation engine that anticipates workflows, triggers event-based actions, and minimizes reactive tasks.
---
Key Features

* Proactive Goal & Task Prioritization: Dynamically evaluates upcoming commitments, deadlines, and user habits to sort tasks by real-world urgency rather than basic chronological order.
* Autonomous Context-Aware Action Plans: Generates personalized, step-by-step productivity recommendations tailored to the user's immediate environment and schedule availability.
* **Intelligent Scheduling & Calendar Assistance: Moves beyond passive alerts by actively structuring focus blocks and suggesting optimal execution windows to prevent last-minute rushes.
* Interactive Conversational Interface: Features an AI assistant capable of processing complex, fluid human inputs to effortlessly update schedules, log habits, and track goals.
* Frictionless Execution Triggers: Designed to convert abstract deadlines into micro-steps, drastically lowering the cognitive barrier to starting high-priority tasks.

Technologies

* Frontend Framework: React / Vite (Structured as a fast, single-page application optimizing UI responsiveness).
* Programming Language: TypeScript (Ensures strongly-typed, predictable, and maintainable code across the application).
* Google AI Studio: Utilized to prototype, fine-tune, and orchestrate the core Gemini-powered prompts that handle conversational understanding, autonomous task planning, and contextual recommendation logic.
* Google Cloud Run: Hosts the containerized application on a fully managed, serverless architecture within the `asia-southeast1` (Singapore) region, ensuring low-latency execution and automatic scaling.
* Google Cloud Artifact Registry: Manages and secures the continuous deployment Docker images for the application service.
